Just came back from a great three-day <a href="http://www.identitycommons.net/">Identity Commons</a> workshop in Sebastopol.  Lots of great people, and five main projects were defined as essential to getting things rolling:

<ol>
  <li>Marketing/Outreach</li>
  <li>Open Source Technology</li>
  <li>Fund raising/Revenue</li>
  <li>Greenhouse/Project Interface</li>
  <li>Legal/Org./Constitution</li>
</ol>

I'm heading up the technology project - gotta set the single sign-on distributed authentication / XRI-lite / profile access management all set up in three months.  Sounds crazy, but for some reason I think it can be done.

Much more came together than I had expected, and now it's up to Owen &amp; Company to see if the big players are going to pony up the big $$.

Too wasted to write much more now - more soon!
